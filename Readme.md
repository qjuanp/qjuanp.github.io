# Working locally

`make run`

# Code structure

Folders:
- data
    - src: scripts to generate data
- content
    - posts
    - public: contains resources relative to the site content (images, js)
- redirects
- site: contains all base html template
    - public: contains reosurces for the tempalte
- plugins: eleventy extensions
